Day Bootstrap Theme For XOOPS CMS 2.5.8.x
=====

![Theme Preview](/theme_preview.png)

Day – a multipurpose HTML template. You can use it for promotion agencies, analysts, marketing, portfolio, and much more. The template is based on Bootstrap framework with the use of modern technology.

Ported to XOOPS [CMS 2.5.8.x](http://xoops.org/modules/news/article.php?storyid=6762)
